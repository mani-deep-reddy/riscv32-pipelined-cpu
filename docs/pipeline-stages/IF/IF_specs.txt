Instruction Fetch (IF) Stage

PC register
    inputs:
        next_PC
        clock
        reset
        PCWrite (stall control)

PC+4 adder
    computes:
        PC_plus4 = PC + 4

PC mux
    selects next PC value from:
        PC_plus4
        branch_target
        jump_target

Instruction memory
    input:
        PC
    output:
        instruction

IF/ID pipeline register
    stores:
        instruction
        PC
        PC+4
    supports:
        write enable
        flush
